*Last Updated: December 2025*

---

## Executive Summary

This document analyzes Detective Sigma's competitive position against Singapore's leading EdTech platforms (Koobits and Geniebook) and identifies opportunities for differentiation and improvement.

**Key Findings:**
1. Detective Sigma has a unique narrative-driven mystery format that competitors lack
2. Missing critical engagement mechanics (daily challenges, reward economy, streaks) that drive daily active usage
3. UI/UX lacks celebration animations, progress visualization, and character mascot that competitors use to boost engagement by 40%+

**What competitors have:**

**What Detective Sigma needs:**

**What competitors have:**
- Koobits: Point system with rewards

### 8.5 Animation Performance Guidelines

**Best Practices to Follow:**
1. Use `transform` and `opacity` only (GPU-accelerated)
2. Add `will-change` hints for complex animations
3. Support `prefers-reduced-motion` media query
4. Lazy load particle systems
5. Limit concurrent animations to 3-4
6. Use `requestAnimationFrame` for scroll effects
7. Test on low-end mobile devices

**Example Reduced Motion Support:**
```css
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```
